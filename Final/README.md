## Введиние

В данной работе была реализована и исследована операция префиксных сумм (inclusive prefix-sum, scan) для одномерного массива целых чисел. Основной целью работы являлось сравнение последовательного и параллельных подходов к вычислению prefix-sum на центральном процессоре и графическом процессоре, а также анализ их производительности и ускорения.

В рамках работы были реализованы три версии алгоритма:

1. Последовательная версия на CPU, использующая прямой проход по массиву;

2. Параллельная версия на CPU с применением технологии OpenMP, основанная на разбиении массива на независимые блоки с последующим добавлением смещений;

3. Параллельная версия на GPU с использованием CUDA, реализующая двухфазный алгоритм сканирования: локальный prefix-sum внутри блоков и последующий перенос сумм блоков ко всем элементам.
